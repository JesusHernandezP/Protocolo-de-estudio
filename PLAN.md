# Los Juegos del DAM - Plan de Desarrollo

## Fase 1: Infraestructura Base
- Configuración de `index.html` y `styles.css`.
- Implementación de `Router.js` (Hash Routing).
- Creación de `DataService.js` para carga de JSON.

## Fase 2: Lógica de Persistencia
- Implementación de `ProgressService.js` (LocalStorage).

## Fase 3: Motor de Quiz
- Desarrollo de `QuizEngine.js` para manejo de estados de pregunta, feedback y puntuación.
- Soporte para "Modo Repaso" (errores específicos).

## Fase 4: Interfaz de Usuario
- Implementación de `UI.js` para renderizado de componentes (Home, Subject, Quiz, Results).

## Fase 5: Contenido y Pulido
- Inserción de preguntas reales para Inglés U1.
- Ajustes de diseño futurista y animaciones CSS.
